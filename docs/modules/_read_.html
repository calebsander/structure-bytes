type

  • type(typeBuffer: ArrayBuffer, fullBuffer?: boolean): Type<any>
    • Defined in read.ts:700

    Deserializes a type, i.e. takes a buffer containing its binary form and creates the type object. The inverse of calling Type.toBuffer.

    Example:

    let type = new sb.ArrayType(
  new sb.FlexUnsignedIntType
)
let typeBuffer = type.toBuffer()
let readType = sb.r.type(typeBuffer)
console.log(readType) // ArrayType { type: FlexUnsignedIntType {} }

    Parameters

    • typeBuffer: ArrayBuffer

      The buffer containing the type bytes

    • Default value fullBuffer: boolean = true

      Whether to assert that the whole buffer was read. In most use cases, this argument should be be omitted.

    Returns Type<any>

    The type that was read

value

  • value<E>(params: ValueParams<E>): E
    • Defined in read.ts:744

    Deserializes a value, i.e. takes the type that serialized it and a buffer containing its binary form and returns the value. Requires the type to be known.

    Example:

    let type = new sb.ArrayType(
  new sb.FlexUnsignedIntType
)
let value = [0, 10, 100, 1000, 10000]
let buffer = type.valueBuffer(value)
let readValue = sb.r.value({type, buffer})
console.log(readValue) // [ 0, 10, 100, 1000, 10000 ]

      The type of value to be read
