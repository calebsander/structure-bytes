Class BooleanTupleType

A type storing a fixed-length array of Boolean values. The length must be at most 255. This type creates more efficient serializations than new sb.TupleType({type: new sb.BooleanType}) for boolean tuples, since it works with bits instead of whole bytes.

Example:

let type = new sb.BooleanTupleType(100)
