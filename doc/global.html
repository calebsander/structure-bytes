download(typeName, options)

(client-side only) Downloads a value using an AJAX request. The type will be sent from the server if not in the client's cache. If the client has already cached the correct type, only the value will be sent. To use, include compiled/download.js or compiled/upload-download.js.
Parameters:
Name Type Description
typeName string An identifier for the type being downloaded. The downloaded type will be cached under this name and future requests that use the same typeName will try to use the type data cached in localStorage. To best utilize the cache, use different type names for requests of different types and use the same type name for requests of the same type.
options Object jQuery AJAX options. These options should specify what URL to use, what to do on success, etc. options.dataType will be overwritten.
Source:
  • client-side/download.js, line 28
Example
sb.download('people', {
  url: '/download-people',
  type: 'GET',
  success: function(value) {
    console.log(value); //e.g. [{name: 'John', id: 2}, {name: 'Jane', id: 10}]
  }
});

httpRespond(params, callbackopt)

Responds to an HTTP(S) request for a value. Will send both type and value if the sig header doesn't match the type's signature. Will only send the value if the signatures match. Response is gzipped to decrease size. Calls callback when done.

upload(params, options)

(client-side only) Uploads a value in an AJAX request. The type used to write the value must be specified. To use, include compiled/upload.js or compiled/upload-download.js.
Parameters:
Name Type Description
params Object
Properties
Name Type Description
type Type The type used to write the value. Only the value is actually sent in the request.
value type The value to upload
options Object jQuery AJAX options. These options should specify what URL to use, what to do on success, etc. options.type or options.method must be set to POST. options.data and options.processData will be overwritten.
Source:
  • client-side/upload.js, line 5
Example
var type = new sb.ArrayType(
  new sb.StructType({
    name: new sb.StringType,
    id: new sb.UnsignedShortType
  })
);
sb.upload({type: type, value: [
  {name: 'John', id: 2},
  {name: 'Jane', id: 10}
]}, {
  url: '/upload-people',
  type: 'POST',
  dataType: 'text',
  success: function(response) {
    alert(response);
  }
});
