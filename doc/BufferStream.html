Class: BufferStream

BufferStream

A class for creating a readable stream out of an ArrayBuffer or a GrowableBuffer. When dealing with very large buffers, this allows chunks to be sent as they are requested rather than stuffing the whole buffer into the stream at once. The stream is intended mainly to be piped into write streams.

Constructor

new BufferStream(buffer)

Parameters:
Name Type Description
buffer external:ArrayBuffer | GrowableBuffer The buffer whose data to use. If a GrowableBuffer is used, only the occupied portion will be written by the stream. Future additions to the GrowableBuffer will not be written. If bytes inside the ArrayBuffer or occupied portion are changed, behavior is undefined.
